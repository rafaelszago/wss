# WSS - Web Services Scan

This tool can be used to identify available services running on a server, it uses NMAP to find out what ports are open.

## Prerequisites

- NMAP (https://nmap.org/)

## Installing

Give execute permission to script:

```bash
chmod +x wss.sh
```

Add script to source:

```bash
source wss.sh
```

## Running

And to run script:

```bash
wss <host> -args
```
